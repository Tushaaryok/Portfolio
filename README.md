Personal Portfolio Website

A modern, responsive, and SEO-friendly personal portfolio website built with **HTML5, CSS3, JavaScript (ES6), Bootstrap 5, and jQuery**. It showcases my skills, projects, experience, and contact information while providing a professional online presence for recruiters, clients, and collaborators.

 🚀 Live Demo

🔗 https://tusharkothariya.vercel.app/

✨ Features

- Responsive design for all devices
- Modern and clean UI/UX
- Smooth scrolling and animations
- Hero, About, Skills, Services, Projects, and Contact sections
- Downloadable resume
- Social media integration
- SEO-friendly structure
- Fast loading and optimized performance
- Cross-browser compatibility

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- jQuery
